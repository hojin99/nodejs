# nodejs

### 환경

* node 14.16.1 이상  

* 개발툴 - Visual Studio Code 사용

### 실행
* npm install을 통해서 의존성 업데이트  
* pacakge.json의 스크립트 이용해서 실행  
* Code Runner를 통해서 Ctrl+Alt+N으로 실행  
* nodemon --watch src/ src/app.js : 소스 변경 시 마다 자동 재실행
